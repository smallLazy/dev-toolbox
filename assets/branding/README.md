# Branding Assets

## App Icon

- **Source**: `src-tauri/icons/icon.png` (128x128 placeholder — needs 1024x1024 source for production)
- **Generation**: `npx tauri icon <source.png>`
- **Output**: `src-tauri/icons/` (multiple resolutions for all platforms)

## Design Guidelines

- **Theme**: Fluent Design dark theme
- **Primary accent**: `#0078D4` (`--color-accent-primary`)
- **Font**: System sans-serif stack (`--font-sans`)
- **Icon style**: Monochrome, 2px stroke, 24x24 viewBox, round caps/joins

## App Branding Icons

All application-level branding icons are defined in `src/design/icons/index.ts` and accessed through `APP_ICONS`:

| Key | SVG | Usage |
|-----|-----|-------|
| `APP_ICONS.toolbox` | Layers3 | Primary brand mark, About page, Dashboard hero |
| `APP_ICONS.workspace` | PanelsTopLeft | Sidebar header logo |
| `APP_ICONS.dashboard` | LayoutGrid | Dashboard section headers |

## Platform Icons

Generated by `npx tauri icon`:
- `icons/32x32.png` — Windows taskbar
- `icons/128x128.png` — macOS Dock (1x)
- `icons/128x128@2x.png` — macOS Dock (2x Retina)
- `icons/icon.icns` — macOS bundle
- `icons/icon.ico` — Windows bundle
- `icons/ios/` — iOS (future)
- `icons/android/` — Android (future)
