# Changelog — Diff Tool

## [1.0.0] - 2026-07-06

### Added
- Initial implementation of Diff tool
- LCS-based line-by-line text comparison
- Unified diff output format
- Context lines control (3, 5, 10, All)
- Ignore whitespace option
- Ignore case option
- Cmd/Ctrl+Enter keyboard shortcut
- Copy output to clipboard
- Clear all inputs and output
- Activity history support
