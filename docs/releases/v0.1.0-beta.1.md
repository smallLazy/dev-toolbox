## 未签名应用说明

本版本**未经过 Apple 代码签名和公证**。在 macOS 上运行时：

1. **首次启动**: 系统可能显示 "无法验证开发者" 警告
2. **手动允许**: 打开 `系统设置 → 隐私与安全性` → 点击 "仍要打开"
3. **或使用命令行**: `xattr -d com.apple.quarantine /path/to/Dev-Toolbox.app`

内测人员需了解此限制。

## 环境限制导致无法自动执行的 GUI 交互项

共 **56 项** GUI 交互功能（按钮点击、文本输入、查看输出）因终端无辅助访问权限无法自动执行。详见 `docs/releases/v0.1.0-beta.1-gui-test.md`。

关键未验证交互路径：
- Base64 中文/emoji 编码解码
- AES CBC/ECB 加密→解密 roundtrip
- MD5/SHA-256 hash 计算
- JWT 解析输出
- 错误 key/IV/hex/base64 前端错误提示
- 配置持久化重启恢复

## 内测反馈方式

内测人员发现问题时，请提供：

- **模块**: 哪个工具面板
- **操作**: 输入了什么，点击了什么
- **预期**: 期望看到什么
- **实际**: 实际看到什么
- **环境**: macOS 版本 + 芯片架构
- **截图**: 如有

反馈模板：

```
模块: Base64
操作: 输入 emoji "🎉" 后点击编码
预期: 得到正确 Base64 编码
实际: 显示乱码 / 报错 / 崩溃
环境: macOS 26.5, arm64
```

## Changelog

### v0.1.0-beta.1 (2026-06-29)

**功能**:
- 🔐 AES-256-CBC / AES-256-ECB 对称加解密（Rust 端，支持 UTF-8/Hex/Base64 编码）
- 🔑 MD5 / SHA-256 哈希计算
- 🔤 Base64 编解码（支持 Unicode 和 Emoji）
- 🔗 URL 编解码（encodeURIComponent / encodeURI）
- ⏰ 时间戳与日期互转（秒/毫秒自动识别）
- 📋 JSON 格式化 / 压缩
- 🎫 JWT Token 解析（Header / Payload / Signature + 过期检测）
- ⚙️ 配置持久化（编码偏好本地存储，重启恢复）

**安全改进**:
- 🔒 CSP 启用 (`default-src 'self'`)
- 🔐 Store 权限最小化 (仅 get/set/save/load)
- 🛡️ ECB PKCS7 padding 严格校验（篡改密文返回明确错误）

**技术改进**:
- 🧹 替换废弃 `escape()`/`unescape()` 为 `TextEncoder`/`TextDecoder`
- 🧪 18 个 Rust 单元测试（含 padding 边界测试）
- 🎯 Clippy 零 warning
- 💥 Tauri 启动失败时优雅退出 (不再 panic)
