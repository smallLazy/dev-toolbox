## 12. 手测结论

- [ ] 通过 — 可打 `v0.1.0-beta.1` 标签，可分发 macOS arm64 内测包给开发人员
- [x] **有条件通过** — 自动化验证全部通过，视觉渲染验证全部通过。**交互式功能（点击按钮、输入文本、查看输出）未在 GUI 层验证**（终端辅助访问权限限制）。记录已知限制后可内测。
- [ ] 不通过 — 存在阻塞性问题，必须修复后重新手测

**备注**: 

**已验证通过（自动化 + 视觉）**:
1. ✅ 应用启动正常（~4 秒，窗口 1100×720，标题 "Dev Toolbox"）
2. ✅ 8 个工具面板全部正常渲染（7 张截图确认）
3. ✅ 侧边栏导航完整显示（8 个菜单项 + 版本号 v0.1.0）
4. ✅ 进程稳定性（运行 3+ 分钟无 crash）
5. ✅ Rust 后端 18/18 单测全绿
6. ✅ 前端构建零错误（npm run build + vue-tsc）
7. ✅ Clippy 零 warning
8. ✅ CSP 已配置 (tauri info 确认)

**需要桌面手动补充验证**:
1. ⚠️ **Base64 编码/解码交互** — 输入→按钮点击→查看输出（TextEncoder/TextDecoder 替换后关键验证）
2. ⚠️ **AES 加密/解密 roundtrip** — 前后端 IPC 交互（Rust 单测已覆盖后端逻辑，但 Tauri invoke 桥接未测）
3. ⚠️ **Hash 计算** — MD5 手写实现需要测试向量确认（特别是空字符串 `d41d8cd...`）
4. ⚠️ **配置持久化重启恢复** — 最关键的内测验证项
5. ⚠️ **JWT 中文 payload 解析** — TextDecoder 替换后的关键验证
6. ⚠️ **错误路径展示** — 错误 key/IV、非法编码、篡改密文等场景的前端错误消息展示
