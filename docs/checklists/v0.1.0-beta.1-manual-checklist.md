### Gate 5: 配置持久化重启恢复

> ⚠️ 这是最关键的内测 Gate。验证 Store 权限最小化后持久化功能仍正常。

| # | 操作 | 预期 | 实际 | ✅/❌ |
|---|------|------|------|-------|
| 5.1 | 打开 配置管理 面板 | 4 个下拉菜单可见 | | |
| 5.2 | 修改 Key 编码为 `Hex`，点击 "💾 保存配置" | 显示 "✅ 配置已保存" | | |
| 5.3 | 修改 IV 编码为 `Base64`，点击保存 | 显示 "✅ 配置已保存" | | |
| 5.4 | 修改输入编码为 `Hex`，点击保存 | 显示 "✅ 配置已保存" | | |
| 5.5 | 切换到 参数加解密 面板，检查编码默认值 | Key=Hex, IV=Base64, Input=Hex, Output=Base64 | | |
| 5.6 | **关闭应用** (Cmd+Q 或关闭窗口) | 应用完全退出 | | |
| 5.7 | **重新启动** `npm run tauri dev` | 应用正常启动 | | |
| 5.8 | 打开 参数加解密 面板，检查编码默认值 | Key=Hex, IV=Base64, Input=Hex, Output=Base64（与 5.5 一致） | | |

**Gate 5 结论**: ⬜ 通过 / ⬜ 不通过
